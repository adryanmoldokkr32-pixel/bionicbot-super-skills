# Brand Asset Creator (AI Designer)

This skill focuses on creating professional visual assets including logos, UI mockups, icons, and marketing illustrations using advanced image generation.

## When to Use This Skill

- Designing a new brand identity.
- Creating hero images for landing pages.
- Generating consistent icons for an app UI.
- Visualizing product concepts.

## Instructions

1. Define the brand's aesthetic (e.g., Minimalist, Tech-futuristic, Playful).
2. Create a detailed prompt for each asset needed.
3. Use `image_gen_v2` with specific parameters (size, quality, background).
4. Ensure visual consistency across multiple generated images.
5. Offer to upload assets to cloud storage or share public URLs.

## Examples

- "Generate a set of 5 minimalist line-art icons for a health app."
- "Create a high-quality hero banner for a futuristic workspace tool."
