# Inbox Zero & Task Manager

This skill helps you reach "Inbox Zero" by categorizing e-mails, drafting replies, and syncing tasks with your calendar.

## When to Use This Skill

- Processing a high volume of unread e-mails.
- Prioritizing urgent messages from clients or managers.
- Drafting quick responses for routine inquiries.
- Turning e-mails into actionable calendar events.

## Instructions

1. List the latest messages in the inbox using integration tools.
2. Categorize each message: [Action Required], [FYI], [Spam], [Newsletter].
3. For [Action Required], suggest a draft reply and a deadline.
4. Offer to schedule follow-ups in the calendar.
5. Provide a summary of what was processed.

## Examples

- "Summarize my unread e-mails and tell me which ones need a reply today."
- "Draft a response to the interview request from John Doe."
